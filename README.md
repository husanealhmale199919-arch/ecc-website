# ECC Website - English Conversation Club

## نادي المحادثة الإنجليزية - جامعة طرابلس

A modern website for the English Conversation Club at Tripoli University, Libya.

## Features

- 🌐 Multi-language support (English, Arabic, French)
- 🔐 Authentication system with role-based access
- 📅 Session management and registration
- 🎫 QR code generation for attendance
- 📊 Admin dashboard
- 📱 Responsive design

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Database**: SQLite / PostgreSQL
- **ORM**: Prisma
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/ecc-website.git

# Navigate to the project
cd ecc-website

# Install dependencies
npm install
# or
bun install

# Set up environment variables
cp .env.example .env

# Initialize database
npx prisma db push

# Run development server
npm run dev
```
